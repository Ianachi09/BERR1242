# Traffic Light System

An enhanced circuit made for the lab. It should be only some buttons, sensors and leds for the lab but I thought why not make a speciallized circuit?

- Randomized timing for green and yellow light.
- Added safety feature (user won't be able to press the button again if the traffic is running).
- Planning to add sensors for cars but Tinkercad runs soo slow with force sensors.
